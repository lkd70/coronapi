# Coronapi

A JavaScript wrapper for Microsoft Bings coronavirus (COVID-19) data.

## Install

```sh
npm install coronapi
```

## Usage

```js
const coronapi = require('coronapi');
coronapi.then(data => {
    console.log(data);
});
```

## Data structure

This module returns a promise with a data object,
this is the Typescript formatted structure of that data object:

```ts
interface IArea {
    totalConfirmed: number,
    totalDeaths: number,
    totalRecovered: number,
    id: string,
    lastUpdated: string,
    areas: [IArea],
    displayName: string,
    lat: number,
    long: number,
    country: string,
    parentId: string,
}
```
